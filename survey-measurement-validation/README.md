# Survey Measurement Validation

A historical statistical analysis comparing customer-experience metrics collected through two independent survey sources.

## What the project demonstrates

- Survey-question and response-scale harmonization
- Data quality checks and missing-value handling
- Overall satisfaction, revisit-intent, and recommendation analysis
- Two-sample Kolmogorov–Smirnov tests
- Welch’s independent-samples t-tests
- Pooled and wave-level comparison of survey samples

## Repository contents

- `survey_measurement_validation.ipynb` — cleaned portfolio version of the analysis

## Data

The original cleaned survey files are not distributed. Brand and source names have been generalized, while the original analytical workflow has been preserved.

## Statistical limitations

Statistical significance is not the same as practical importance. In addition, failure to reject a difference test does not establish equivalence. A production validation study should also report effect sizes, confidence intervals, multiple-testing considerations, and predefined equivalence bounds.
