# Resume Portfolio

A curated collection of analytics, data science, and machine learning work selected to demonstrate practical problem solving, technical depth, and clear business communication.

Projects are organized into separate folders and may include notebooks, documentation, sample or synthetic data, visualizations, and supporting scripts. Each project README describes the objective, methods, assumptions, and limitations.

## Areas of Focus

- Exploratory and diagnostic analysis
- Statistical modeling and machine learning
- Customer, retention, and cohort analytics
- Forecasting and decision support
- Data preparation, feature engineering, and model evaluation
- Reproducible analytical workflows

## Tools

Python, SQL, pandas, NumPy, scikit-learn, matplotlib, Jupyter, and related analytics tooling.

## Data and Confidentiality

Projects in this repository use public, synthetic, anonymized, or assessment-provided data. Proprietary data, personally identifiable information, credentials, and employer-specific infrastructure are not included.
