# Customer Retention Analysis

An end-to-end analysis of subscription customer behavior using assessment-provided, cleaned member and transaction data.

## Objective

The analysis evaluates monthly customer and revenue retention, estimates customer longevity, and compares performance across business segments. The focus is on translating transactional data into practical retention metrics and segment-level insights.

## Methods

- Monthly revenue, order, and active-customer aggregation
- New-versus-existing customer classification
- Month-over-month customer retention and churn
- Revenue retention and revenue at risk
- Customer longevity and purchase behavior
- Segment comparisons by industry, company size, and region

## Files

- `customer_retention_analysis.ipynb` — cleaned and reorganized analysis

## Data

The original assessment supplied two cleaned CSV files:

- `DA_Project_Member_Data.csv`
- `DA_Project_Revenue_Data.csv`

Those files are not redistributed in this portfolio. The notebook documents the expected fields and includes validation checks so the analysis can be rerun when compatible data is available.

## Notes

This project was originally completed in 2019 and has been updated for readability, reproducibility, and current pandas/matplotlib practices. The analytical approach is preserved while assessment-specific instructions and presentation language have been removed.
